# 22 — Carmel/Hamilton Compliance and Public Finance Playbook

> Clay Terrace-focused workflow for moving Carmel/Hamilton public-finance and compliance items from possible to verified.  
> Diligence and operations guide only; legal and tax conclusions require qualified review.

## Scope

- Public-finance tracks: redevelopment support, bonds, TIF, tax abatement, tourism-linked support.
- Compliance tracks: permits, code, zoning, fire/life safety, utility and engineering approvals.

## Public Finance Workflow

1. Confirm district and project fit
- parcel and address mapping
- district/zoning confirmation
- redevelopment or TIF relevance

2. Build support package
- city/county support letter target
- development narrative and public-benefit case
- budget and capital stack with obligations separated

3. Secure formal records
- public-body resolutions/hearing records
- bond/TIF term-sheet evidence
- counsel and tax/accounting memos

4. Lender-safe gating
- mark as `needs_review` until approvals exist
- mark as `application_ready` only after submission evidence
- mark as `awarded` only when official approval terms are documented
- mark as `verified` only after compliance and accounting treatment are complete

## Carmel Permit Workflow

Core sequence:

- project setup and zoning review
- permit application package
- plan review and discipline comments
- revisions and resubmittals
- permit issuance
- inspection scheduling and approvals
- close-out and certificate path

Evidence expected:

- submitted permit applications
- plan-review status and comments
- inspection records
- final approvals/certificates

## Indiana Code Workflow

Required tracks:

- Indiana building code
- Indiana fire code
- Indiana mechanical code
- Indiana fuel gas code
- Indiana electrical code
- Indiana energy conservation code
- ADA/accessibility
- elevator/lift review (if applicable)
- stormwater and engineering approvals
- sign permits
- health/food service/pool-spa reviews where applicable

Blocking rule:

- unresolved required compliance items block lender-ready status.

## Hotel Tax and Operating Tax Controls

Obligation classes:

- Indiana accommodations sales tax
- Hamilton County innkeeper tax
- local food and beverage tax (where applicable)
- property-tax and assessment review
- sales/use tax treatment for materials and FF&E
- payroll withholding and related state tax handling

Control rule:

- obligations are tracked for readiness and risk, never counted as funding.

## Evidence to Move from Possible to Verified

Minimum move-to-verified stack:

- official source citations for each program
- project address and parcel mapping
- zoning and utility confirmation
- permit and code evidence chain
- IEDC/public-body approval records where applicable
- tax/accounting memo
- legal memo
- human approval log

Without this stack, items remain `possible`, `needs_review`, `monitor_only`, or `blocked`.
