## Platform Rule

Taxes and code requirements are obligations, not funding sources.

## Lender-Safe Funding Treatment

- `estimated`: possible only, not verified funds
- `application_ready`: package drafted but not submitted
- `submitted`: pending review/award
- `awarded`: still separate from verified funds until all conditions and accounting treatment are completed
- `verified`: may count toward verified funding after evidence and review
- `not_counted`: obligations and monitor-only items

Never combine these stages into a single "funding available" number.
